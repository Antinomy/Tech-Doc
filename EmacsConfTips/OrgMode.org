


* 文档元数据

具体的内容可以到文档中查看：Export options 我们除了手动添加上面的元数据外，
还可以用快捷键 C-c C-e t 插入选项，其中可能有些选项我们需要经常用到：

|----------+----------------------------------|
| H        | 标题层数                         |
| num      | 章节(标题)是否自动编号           |
| toc      | 是否生成索引                     |
| creator  | 是否生成 "creat by emacs…"       |
| LINKUP   | UP                          链接 |
| LINKHOME | HEME                        链接 |
|----------+----------------------------------|

e.g.

#+OPTIONS: num:nil
